Supply Chain Analytics: Correlation Matrix

Analyst Email: 23f2000790@ds.study.iitm.ac.in

Project Overview

This repository contains the analysis of supplier performance data from 59 procurement transactions. The goal is to identify relationships between key supply chain metrics such as Lead Time, Inventory Levels, and Cost per Unit.

Files

correlation.csv: The calculated correlation matrix for the supply chain variables.

heatmap.png: A visualization of the correlation matrix using a Red-White-Green color scale (Red = Low, Green = High).
